# AI-ML-Django-Projects

## Overview
This repository contains Machine Learning web applications developed using Django and Python.

## Projects Included

### Resume Analyzer AI
- Extracts skills from resumes
- Matches resumes with job descriptions
- Uses NLP, TF-IDF, and Cosine Similarity

### Crop Recommendation System
- Recommends suitable crops based on soil nutrients and environmental conditions
- Built using Machine Learning algorithms

### Cinnamon Quality Classification
- Predicts cinnamon quality using Machine Learning
- Includes data preprocessing, feature engineering, and model evaluation

## Technologies Used
- Python
- Django
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
